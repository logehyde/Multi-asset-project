# Dynamic Multi-Asset Allocation Project - Starter Template

This repository contains starter Python modules and a structure for a multi-asset research project:
* Dynamic Multi-Asset Allocation Model Using Macro Regimes and Cross‑Asset Signals

```

├─ src/

│  ├─ regimes.py
│  ├─ signals.py
│  ├─ portfolio.py
│  ├─ backtest.py
│  ├─ stress.py
│  ├─ attribution.py
│  ├─ reporting.py
│  └─ utils.py
├─ reports/                # generated reports and decks
├─ requirements.txt
└─ run_project.py
```
